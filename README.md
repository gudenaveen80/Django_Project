# 🚗 CarZone

A car dealership website I built using Django. You can browse cars, 
contact the team, and make inquiries — all managed through an admin panel.

## 🌐 Live Site
👉 (https://django-project-7vty.onrender.com)

## 🔧 What I Built
- Car listings with search and filter
- User signup and login system
- Contact form that saves messages to the database
- Car inquiry form for interested buyers
- Admin panel to manage everything

## 🛠️ Built With
- Django & Python
- HTML, CSS & Bootstrap
- SQLite (local) / PostgreSQL (production)
- Hosted on Render

## 💻 Run it locally
```bash
git clone https://github.com/gudenaveen80/Django_Project.git
cd Django_Project
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
